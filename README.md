Este projeto foi desenvolvido como parte de um desafio técnico para a TDS03corp. O objetivo foi criar uma interface funcional e semântica para um Clube do Livro, focando em usabilidade, criatividade e organização de código.
🚀 Tecnologias Utilizadas

    HTML5: Uso de tags semânticas para melhor acessibilidade e SEO.

    CSS3: Estilização moderna utilizando Variáveis CSS (root:) para manter a consistência da paleta de cores.

    JavaScript (opcional): Para lógica de interatividade.

🛠️ Requisitos Implementados

Conforme solicitado no edital, o projeto conta com:

    Navegação entre Páginas: Estrutura com pelo menos duas páginas distintas (ex: Home e Catálogo).

    Elementos Interativos:

        Exemplo: Card de livro com efeito hover (escala).

        Exemplo: Botão de "Ver Detalhes" que revela conteúdo oculto.

    HTML Semântico: Uso rigoroso de tags como <header>, <main>, <section>, <article> e <footer>.

    Variáveis CSS: Centralização das cores do projeto para fácil manutenção.

    Organização de Arquivos: Pastas separadas para /assets, /css, /js e /pages.

✒️ Decisões de Design e Código

    Legibilidade: O código foi escrito pensando no trabalho em equipe, com nomes de classes intuitivos e comentários explicativos onde necessário.

    Usabilidade: O design foi pensado para que o usuário encontre rapidamente as informações sobre as reuniões e livros do clube.

💡 Dicas para você brilhar no teste:

    O Nome do Clube: Como o teste permite escolher, use algo criativo como "Capítulo Final", "Folha de Outono" ou "Libri Mundi".

    Variáveis CSS: No topo do seu arquivo CSS, não esqueça de declarar!

    Semântica: Use <article> para cada livro listado e <section> para separar blocos como "Quem Somos" ou "Próximas Leituras".
